## 拥抱云时代容器化微服务化系统架构

* 云化

    云计算时代系统设计大部分开始转向云化更合理配置的利用云服务资源。

* 容器化

    以Docker为代表的容器化技术大潮席卷整个互联网行业。

* 微服务化

    容器化的企业架构，基本可以通过少量重构完成微服务化，降低成本提高可用性。
